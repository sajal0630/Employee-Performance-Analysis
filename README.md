# Employee-Performance-Analysis

📊 Employee Performance & Satisfaction Analysis
🚀 Project Overview

This project focuses on analyzing employee performance and satisfaction using Google BigQuery and Looker Studio.

The goal was to transform raw HR data into meaningful insights that can help organizations understand how factors like education, experience, and job role influence employee performance.

🎯 Objectives

Analyze employee performance trends across different segments

Understand the relationship between education level, job role, and salary

Identify patterns in employee satisfaction and tenure

Build an interactive dashboard for business insights

🗂️ Dataset Description

The analysis was performed using two datasets:

1. Employee Dataset

EmployeeID

EducationLevel

Age

Salary

Department

JobRole

Gender

YearsAtCompany

YearsWithCurrManager

YearsSinceLastPromotion

2. Performance Dataset

PerformanceRating

RatingLevel

SatisfactionLevel

🛠️ Tech Stack

Google BigQuery – Data cleaning, transformation, SQL queries

SQL – Joins, aggregations, grouping

Looker Studio – Dashboard creation and visualization

⚙️ Data Processing Steps
🔹 Data Cleaning

Handled missing and inconsistent values

Standardized categorical variables

🔹 Data Integration

Joined Employee and Performance datasets using SQL

🔹 Data Transformation

Aggregated data using GROUP BY

Created metrics for analysis (counts, distributions, averages)

📈 Dashboard Highlights

The interactive dashboard provides insights into:

Employee distribution across departments and job roles

Salary distribution by department, education, and demographics

Performance and satisfaction trends

Employee tenure and promotion patterns

💡 Key Insights

Majority of employees are concentrated in the Technology department

Employees with longer tenure tend to show more stable performance patterns

There is a noticeable gap between time at company and promotions, indicating potential growth opportunities

Salary varies significantly across roles, education levels, and departments

Employee performance is influenced by a combination of experience, role, and satisfaction levels
